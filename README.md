XPreferences
============
XPreferences is a AppleScript Library to help saving and loading settings(preferences) of an AppleScript application.

## Usage
http://www.script-factory.net/XModules/XPreferences/manual/index.html

## Building
Reqirements :
* OS X 10.9 or later.
